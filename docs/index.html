Peapod 🫛

Sandboxes isolados e descartáveis para agentes de IA — local, no seu Mac.

Ver no GitHub Começar

Por quê

Deixar uma IA rodar código na sua máquina é risco — um bug ou uma dependência maliciosa toca seus arquivos, sua rede, suas chaves. O Peapod aplica privacidade e menor privilégio a esse problema: cada execução fica isolada, descartável, com rede desligada por padrão e uma trilha de auditoria completa de tudo que rodou.

Funcionalidades

Feito para agentes

Dirigido por MCP — agentes criam, rodam e descartam sandboxes sozinhos (12 ferramentas).

Trilha de auditoria

Todo comando que um agente roda fica registrado e revisável.

Firewall por domínio

Proxy de allowlist de egresso — o sandbox alcança só os domínios que você liberar.

Snapshots

Snapshot, fork, prune e diff do filesystem de um sandbox.

Isolamento plugável

Docker/Podman, ou uma microVM por sandbox via o framework container da Apple.

App nativo + web

Um app SwiftUI de macOS e um dashboard web, sobre o mesmo núcleo.

Começar

go build -o bin/peapod ./cmd/peapod

ID=$(bin/peapod sandbox create python:3.12-slim --net none)
bin/peapod sandbox exec "$ID" python3 -c 'print(6*7)'
bin/peapod sandbox history "$ID"
bin/peapod sandbox rm "$ID"

# app nativo:
cd ui-native && ./build.sh   # → Peapod.app + Peapod.dmg
